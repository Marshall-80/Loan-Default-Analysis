# Loan Default Analysis

## Project Overview

This project analyzes loan application and repayment data to identify patterns that contribute to loan defaults. The goal is to help financial institutions make better lending decisions by using SQL for data analysis and Power BI for interactive reporting.

---

## Business Problem

Loan defaults can significantly impact a bank's profitability. This project aims to identify high-risk customer segments and provide data-driven insights that can improve loan approval strategies and reduce default rates.

## Key Insights

- Customers with higher DTI ratios were more likely to default.
- Debt Consolidation accounted for the largest share of issued loans.
- Certain regions had noticeably higher default rates.
- Higher-grade loans showed better repayment performance.

---

## Business Recommendations

- Strengthen approval criteria for applicants with high DTI ratios.
- Apply additional risk monitoring to high-risk customer groups.
- Use regional trends to improve lending strategies.
- Review pricing and approval policies for higher-risk loans.
